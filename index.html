A Petrophysics Workstation in a Single HTML File

Petrolog Studio loads one or many LAS files, renders professional multi-track well logs, correlates formation tops across wells, builds user-defined zones, runs statistical analysis, and produces three distinct QC reports — all without installing anything. Open the file in any modern browser and you are working.

Core Capabilities

📊

Multi-Well Cross-Section

Load any number of LAS files and render them side by side with configurable tracks, color, scale, and line styles per curve.

📍

Formation Tops

Import tops from CSV, label each formation once per well, and draw Petrel-style diagonal correlation lines between adjacent wells.

🎨

Zones & Fills

Define coloured intervals between any two formation tops. Fills render across wells and break cleanly at missing tops.

📈

Statistics & Analysis

Per-well per-curve stats table, histograms with configurable bins, and crossplots with linear regression & R².

✓

Log QC & Coverage

Visual data-completeness grid, range validation (GR, RHOB, NPHI, RT), and depth monotonicity / gap / duplicate checks.

🎯

Formation Tops QC

Wells × formations matrix showing which tops are present or missing, with summary scores and by-formation / by-well breakdowns.
